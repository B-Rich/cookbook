For any x,
$$\mathrm{sin(x)} = \left ( \frac { 2 \mathrm{x}}{\pi} \right ) \left ( \frac { 4 - \mathrm{y}}{3} \right ) \left ( \frac { 16 - \mathrm{y}}{15} \right ) \left ( \frac { 36 - \mathrm{y}}{35} \right ) ... \left ( \frac { (2 \mathrm{n})^2 - \mathrm{y}}{(2 \mathrm{n})^2 - 1 } \right ) ...$$
where $ \mathrm{y} = \left ( \frac{2 \mathrm{x}}{\pi} \right ) ^2 . $
