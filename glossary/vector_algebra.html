The following set of rules are called the rules of vector algebra:

addition:

if x = (x1, x2, ..., xn) and y = (y1, y2, ..., yn),  then x + y = (x1 + y1, x2 + y2, …, xn + yn); 

scalar multiplication:

if x = (x1, x2, ..., xn) is a vector and t is a scalar, then tx = (tx1, tx2,  ..., txn).

A closed set of objects obeying these rules is a vector space. A single such object is a vector.
