Let S be a non-empty collection of sets. A non-empty subset C of S is said to be a chain if for any A and B  ∈ C, either A ⊆ B or B  ⊆ A.

Suppose that whenever C is a chain in S, then ⋃D ∈ C is an element in S. Then S contains a maximal element.

Or, more generally, if  ≤  is a partial order on a non-empty set S and every chain in S has an upper bound, then S has a maximal element.
