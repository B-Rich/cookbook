A transformation which is useful for circularly symmetric functions, i.e. functions which depend only on the distance from the origin. The nth order Hankel transform of a function f(r) is
Fn(s) ≡ ∫0∞rf(r)Jn(rs)dr, 
where Jn is an nth order Bessel function.

The Hankel transform of order zero is equivalent to a 2-D Fourier transform of the same function.

The inverse of a Hankel transform is as follows:
f(r) ≡ ∫0∞sFn(s)Jn(rs)ds.
