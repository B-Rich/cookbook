Given a linear differential operator L , the Green’s function G(x, xi) solves
LG(x, xi) = δ(x − xi); 
then to solve Ly = f(x) we apply the operator ∫dxi, G(x, xi) to find y(x) . If L is a second order linear operator, then G is continuous in the first variable at x = xi and [G′]xi − xi +  = 1 at x = xi .
