If f is a convex function, then for any sequence x1, x2,  ..., xn : 
$$\mathrm{f} \left ( \frac {\mathrm{x}_{1} + \mathrm{x}_{2}+ ... + \mathrm{x}_{\mathrm{n}} }{\mathrm{n}} \right ) \leq \frac{ \mathrm{f}( \mathrm{x}_{1}) + \mathrm{f}( \mathrm{x}_{2}) +... + \mathrm{f}( \mathrm{x}_{ \mathrm{n} } ) }{ \mathrm{n}} .$$
If f is concave, the inequality is the other way around.
