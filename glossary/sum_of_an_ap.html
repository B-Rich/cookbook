If an arithmetic progression has first term a and common difference d, then the sum of the first n terms of the progression is $ \frac{1}{2} \mathrm{n} ( 2 \mathrm{a} + ( \mathrm{n} - 1 ) \mathrm{d} ) . $

In other words,
a + (a+d) + (a+2d) + ... + (a+(n-1)d) = ∑i=0n − 1a+id

$$= \frac{1}{2} \mathrm{n} ( 2 \mathrm{a} + ( \mathrm{n} - 1 ) \mathrm{d} ) .$$
