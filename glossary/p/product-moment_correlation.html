A sample (xi, yi) of size n has Pearson’s coefficient
$$r = \frac{\sum ( x_i - \bar{x} ) ( y_i - \bar{y} )} {\sqrt{ \sum ( x_i - \bar{x} ) ^2 \sum ( y_i - \bar{y} ) ^2 } } = \frac{ \frac{\sum x_i y_i }{n} - \bar{x} \bar{y} } { \sqrt { \left ( \frac{\sum x_i ^2 }{n} - \bar{x} ^2 \right ) \left ( \frac{\sum y_i ^2 }{n} - \bar{y} ^2 \right ) } } .$$
