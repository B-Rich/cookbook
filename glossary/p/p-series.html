The p - series is:
$$1 + \left ( \frac {1}{2} \right ) ^{\mathrm{p}} + \left ( \frac {1}{3} \right ) ^{\mathrm{p}} + \left ( \frac {1}{4} \right ) ^{\mathrm{p}} + \left ( \frac {1}{5} \right ) ^{\mathrm{p}} + ...$$

If p=1, it is the harmonic series.

If p>1, the series converges; otherwise it diverges.
