A function f : Z2 → Z of the form f(x, y) = ax2 + bxy + c for some a, b, c ∈ Z . We define the discriminant of f to be Δ(f) = b2 − 4ac . If f is positive-definite (i.e. it takes strictly positive values except for f(0, 0) = 0 ) then Δ(f) < 0 , and if a, c > 0 and Δ(f) < 0 then f must be positive-definite. Similarly, f is negative-definite iff a, c < 0 and Δ(f) > 0 .
