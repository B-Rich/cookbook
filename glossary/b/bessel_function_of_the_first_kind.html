The Bessel function of the first kind of order n is written Jn(z),  and defined as
$$\mathrm{J}_{\mathrm{n}}\mathrm{(z)} = \frac{1}{\pi} \int_0^{\pi} \mathrm{cos(nt - zsin(t))dt}$$

where n is not zero.

If n is a positive integer, this is equal to:
$$\mathrm{J}_{\mathrm{n}}\mathrm{(z)} = \sum_{\mathrm{r}=0}^{\infty} \frac{(-1)^{\mathrm{r}}} {\mathrm{r!} \Gamma \mathrm{(n+r+1)}} \left ( \frac{\mathrm{z}}{2} \right ) ^ { \mathrm{n+2r}}$$

These functions arise in the solution of the wave equation in cylindrical coordinates. They satisfy Bessel’s equation.
