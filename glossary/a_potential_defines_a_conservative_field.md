If a potential exists for a given field, then that field is
conservative. Proof: Let f(x) be a field with potential
!['  //phi ( x ) ,  '](../dictionary/equation_images/3471.8..png) and
let C be a curve. Then: \\
!['  the integral of  \_(C) f.dx  = -  the integral of  \_(C) //nabla //phi . dx '](../dictionary/equation_images/3471.1..png)
\\
![' = -  the integral of  \_(t\_0) \^(t\_1) //nabla //phi .   (dx)/(dt) dt '](../dictionary/equation_images/3471.2..png)
\\
![' = -  the integral of  \_(t\_0) \^(t\_1)   ( (d(partial) //phi)/(d(partial) x) (dx)/(dt)  + (d(partial) //phi)/(d(partial) y) (dy)/(dt)  + (d(partial) //phi)/(d(partial) z) (dz)/(dt)   ) dt '](../dictionary/equation_images/3471.3..png)
\\
![' = -  the integral of  \_(t\_0) \^(t\_1)  (d)/(dt) //phi ( x(t),y(t),z(t) ) dt '](../dictionary/equation_images/3471.4..png)
\\
![' = -  the integral of  \_(t\_0) \^(t\_1)  (d)/(dt) //phi ( x(t) ) dt '](../dictionary/equation_images/3471.5..png)
\\
![' = //phi ( x ( t \_0 ) )   - //phi ( x ( t \_1 )
) '](../dictionary/equation_images/3471.6..png)
\\
![' = //phi ( x \_0 ) - //phi ( x \_1 ) '](../dictionary/equation_images/3471.7..png)
So the value of the integral depends only on the endpoints of C, and not
on the shape of C. In particular, if C is a closed curve then this
integral is equal to zero - so f(x) is conservative.
