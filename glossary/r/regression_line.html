A sample (xi, yi) of size n has least squares regression line of y on x:
y − ȳ = b(x − x̄), 
where
$$b = \frac{\sum (x_i - \bar{x} ) (y_i \bar{y} ) } { \sum ( x_i - \bar{x} ) ^2 } = \frac{\sum \frac{x_i y_i}{n} - \bar{x} \bar{y} } {\sum \frac{x_i ^2 }{n} - \bar{x} ^2 } .$$
This is a line of best fit for these data.
