An equivalence relation  ∼  defined on a set S is:

  • reflexive: a ∼ a,

  • transitive: a ∼ bandb ∼ c ⇒ a ∼ c, and

  • symmetric: a ∼ b ⇒ b ∼ a

for all a, b, and c in S.

Given an equivalence relation on a given set, then the equivalence class of an element a is the set of elements which are equivalent to a under that relation.

The equivalence classes form a partition of the set.
