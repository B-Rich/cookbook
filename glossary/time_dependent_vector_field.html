A time dependent vector field on U is a map
f : ( − ε, ε) × U → Rn
which associates a vector f(t, x) to each time t and point x .

Let x0 ∈ U . An integral curve for the vector field f with starting point x0 is a map
ϕ : ( − δ, δ) → U
such that
$$\frac{d \phi}{dt}=f(t, \phi(t)) \quad \text{and} \quad \phi(0)=x_{0}$$
so that the tangent vectors to ϕ are just the values of the vector field at that point in time.
