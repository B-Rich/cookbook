If we differentiate something twice, then the result of this is the second derivative. It is written $ \frac{\mathrm{d}^2 \mathrm{y}}{\mathrm{dx}^2}. $

So for instance, if y = 5x3 + 6x2 + x + 4, 
$$\frac{\mathrm{dy}}{\mathrm{dx}} = 15 \mathrm{x}^2 + 12 \mathrm{x} + 1$$

$$\frac{\mathrm{d}^2 \mathrm{y}}{\mathrm{dx}^2} = 30 \mathrm{x} + 12$$
