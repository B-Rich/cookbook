A power series of the form:
f(x + a) = f(a) + fʹ(a)x + fʺ(a)x2/2 + f‴(a)x3/3! + ... + f(n − 1)(a)xn − 1/(n − 1)! + Rn
is called a Taylor series. Taylor’s theorem states the conditions under which such a function will exist for a function f. Rn is a remainder term; it can be written in various forms; its value is usually small.
