If we differentiate an implicit equation without first making it explicit, this is called an implicit differentiation. Very often we have to use the product rule in order to do this.

For example: xy = 2

Differentiate with respect to x, using the product rule: $ \frac{\mathrm{d}}{\mathrm{dx}} (\mathrm{xy}) = 0 $
$$\frac{\mathrm{dx}}{\mathrm{dx}} \mathrm{y} + \mathrm{x} \frac{\mathrm{dy}}{\mathrm{dx}} = 0$$

$$\mathrm{y} + \mathrm{x} \frac{\mathrm{dy}}{\mathrm{dx}} = 0$$
