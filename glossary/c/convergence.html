A property of some sequences. A sequence is called convergent if it converges, i.e. a sequence ui is convergent if there exists a value u with the property that by choosing a large enough value of i, we can make ui as close as we wish to u.

Formally, ui is convergent if:
∃u : ∀ε > 0, ∃n ∈ N such that ∀i ≥ n, ∣ui − u∣ ≤ ε.
