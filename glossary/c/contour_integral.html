If we have a function f defined in the complex plane, and a curve C in the complex plane, then the integral of f along C is called a contour integral and is written ∫Cf(z)dz.

Such an integral can be defined as follows: dissect C into n segments with endpoints x0...xn.Pickapointzi in each segment. The integral is the limit, if it exists, of the sum ∑f(zi)(xi − xi-1),  as n approaches infinity and the length of the longest segment goes to zero.
