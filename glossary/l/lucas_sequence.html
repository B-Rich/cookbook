Either: The sequence beginning 1, 3... in which each number is equal to the sum of the previous two. It is closely related to the Fibonacci sequence. Or (less usually): The sequence of numbers un defined by:
un = 4un − 1 − un − 2.
Usually we take the first two values as u0 = 2,  u1 = 4,  so the sequence is:
2, 4, 14, 52, 194, 724, 2702, 10084, 37634...
Values in this sequence all satisfy the equation s2 − 3r2 = 4 for an integer value of r.
