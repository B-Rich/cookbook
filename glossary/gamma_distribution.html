$$\mathrm{f(x)} = \frac{\mathrm{x}^{\mathrm{a} -1} \mathrm{e}^{- \mathrm{x/b}}} { \mathrm{b}^{\mathrm{a}} \Gamma ( \mathrm{a} ) } .$$
