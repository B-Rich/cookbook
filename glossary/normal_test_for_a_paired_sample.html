This is a test of the null hypothesis that the difference in two population means is μ,  given a paired sample (xi,  yi) of size n. The distribution used is N(0,1), and the test statistic is $ \frac{ ( \bar{x} - \bar{y} ) - \mu } {\sigma / \sqrt{n} } . $ This test is identical to the normal test for a mean, treating the differences xi − yi as a single sample.
