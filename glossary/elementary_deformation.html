A path ψ is said to be an elementary deformation of φ (both with domain [a, b] ) if we can find a = x0 < x1 < … < xn = b and convex open subsets C1, …, Cn of D such that, whenever t ∈ [xi − 1, xi] , we have both φ(t) ∈ Ci and ψ(t) ∈ Ci . For a holomorphic function f , we then have
∫ψf(z)dz = ∫φf(z)dz.
