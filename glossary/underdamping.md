A damped harmonic system
!['  (d\^2 x)/(dt\^2) + //alpha (dx)/(dt)   + //omega \^2 x = 0   '](../dictionary/equation_images/2100.1..png)
is described as underdamped if
!['  //alpha \< 2 //omega .  '](../dictionary/equation_images/2100.2..png)

In this case the equation is solved by
!['  x = a e\^( - //alpha t / 2 ) //sin ( //rho t + epsilon ) ,  '](../dictionary/equation_images/2100.3..png)
where
!['  //rho = (1)/(2)  the square root of  ( 4 //omega \^2 - //alpha \^2 ) .  '](../dictionary/equation_images/2100.4..png)

The system oscillates forever, the amplitude of the oscillations
becoming exponentially smaller over time.
