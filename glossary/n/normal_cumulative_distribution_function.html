The cumulative distribution function of a normal variable is:
$$\mathrm{F(x)} = \int _{- \infty} ^{\mathrm{x}} \frac{1}{\sqrt{2 \pi \sigma ^2 }} \mathrm{e}^{- ( \mathrm{t} - \mu ) ^2 / 2 \sigma ^2 } \mathrm{dt}$$

$$= \int _{- \infty} ^ {( \mathrm{x} - \mu ) / \sigma} \frac{1}{\sqrt{2 \pi}} \mathrm{e} ^ {- \mathrm{t} ^2 / 2 } \mathrm{dt}$$

We call this $ \Phi \left ( \frac { \mathrm{x} - \mu } { \sigma } \right ) . $ We have Φ(x) → 0 as t  →  − ∞,  and Φ(x) → 1 as t → ∞. Tables of values of Φ are often given in statistics textbooks.
