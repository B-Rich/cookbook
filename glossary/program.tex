A list of instructions given to a computer to carry out a particular task.

\par
For example, A LOGO program to draw a square of size 20 is:
\par

\texttt{
Forward 20; 
Right 90;   
Forward 20; 
Right 90;  
Forward 20;
Right 90;   
Forward 20;
}

Alternatively, one might say:
\texttt{REPEAT 4 [Forward 20; Right 90]}
