For independent random variables X and Y, the property var(X + Y) = var(X) + var(Y) holds true.

X is binomially distributed if X = ∑r = 0nXr and each Xr is either 1 or 0 depending on the success or failure of n independent Bernouilli trials. If the probability of success is p, it follows that:
var(Xr) = p − p2 = p(1 − p)
Since the Xr are independent, we can conclude:
var(X) = np(1 − p).

Alternatively, using the notation ⟨X⟩ to indicate the expectation of the random variable X, we have ⟨X⟩ = np if X is binomially distributed with parameters n and p. And:


$$\begin{aligned} \langle X ^2 \rangle &=& \sum_{r=0} ^{n} r^2 p_{r} \\ &=& \sum_{r=0}^{n} r(r-1) {n \choose r} p^{r} (1-p)^{n-r} + \sum _{r=0}^{n} r {n \choose r} p ^{r} (1-p)^{n-r} \\ &=& n(n-1)p ^2 + np\\ \mbox{So,} var(X) &=& \langle X ^2 \rangle - \langle X \rangle ^2 \\ &=& n(n-1)p ^2 + np - n^2 p^2 \\ &=& n^2 p^2 - np^2 + np - n^2 p^2 \\ &=& np - np^2 \\ &=& np(1-p)\end{aligned}$$
